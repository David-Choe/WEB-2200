Analog Input

Github

For this second half of the lab adding the analog controller was more difficult than expected. Although coding the Arduino was not bad I ran into an issue where the dc motor would just refuse to do the analog changes sometimes. While I wanted to change the speed of the fan with the potentiometer it only like being in the max speed or off-state it refused to work in the in-between. So I switched to changing the direction of the fan which worked out well, but sometimes the motor just didn’t like switching and sometimes didn’t do it. In the end, I got it to work but I think there are some issues that still exist.
